pyquantus.qus.analysis module

class pyquantus.qus.analysis.SpectralAnalysis

Bases: object

Complete QUS analysis of an ultrasound image given a corresponding phantom image.

This class supports both scan converted and non-scan converted images. QUS analysis is performed on each window generated by the generateRoiWindows method. Spectral analysis corresponds to the computation of the midband fit, spectral slope, and spectral intercept parameters using the frequency domain of each window.
