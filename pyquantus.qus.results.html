pyquantus.qus.results module

class pyquantus.qus.results.SpectralData

Bases: object

Class to store spectral data and images after QUS analysis.

This class supports both scan converted and non-scan converted images. Once analysis is completed in the SpectralAnalysis class, the results are stored in this class for further evaluation and visualization.

plotPsData()

Plots the power spectrum data for each window in the ROI.

The power spectrum data is plotted along with the average power spectrum and a line of best fit used for the midband fit, spectral slope, and spectral intercept calculations. Also plots the frequency band used for analysis.
